# Tooltip

Display a **tooltip** attached to any kind of element with different positioning.

It can display number or strings.

<Playground>
  <Button
    badgeContent="pending"
    badgeColor="warning"
    as={Tooltip}
    tooltipContent="Tooltip Text"
  >
    Notifications
  </Button>
</Playground>

## API

<ForwardRefAsExoticComponentDoc
  component={Tooltip}
  customize
  docUrl="https://wikiki.github.io/elements/tooltip/"
  docProvider="Bulma-Extensions"
  props={{
    tooltipContent: {
      description: "the contents of the tooltip",
      typeName: "number | string",
    },
    tooltipColor: {
      description: "the color of the tooltip",
      typeName: "string",
      typeTip: mapEnumerable(DEFAULTS.colors),
    },
    tooltipMultiline: {
      description: "force multiline display",
      typeName: "boolean",
    },
    tooltipPosition: {
      description: "the position of the tooltip",
      typeName: "string",
      typeTip: mapEnumerable(TOOLTIP_DEFAULTS.positions),
    },
  }}
/>
