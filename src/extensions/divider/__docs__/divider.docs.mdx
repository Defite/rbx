# Divider

Display a vertical or horizontal **divider** to segment your design.

### Colors

Use the `color` prop on `<Divider>` to specify the color.

<Playground>
  {() =>
    DEFAULTS.colors.map((color, i) => (
      <OptionBlock props={{ color }} index={i} key={i}>
        <Divider color={color}>OR</Divider>
      </OptionBlock>
    ))
  }
</Playground>

### Vertical

Use the `vertical` prop of `<Divider>` to make it vertical.

<Playground>
  <Divider vertical>OR</Divider>
</Playground>
