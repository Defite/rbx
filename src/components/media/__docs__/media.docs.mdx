# Media Object

The famous **media object** prevalent in social media interfaces, but useful in any context.

The example below adds a `1px solid black` border to each `<Media.Item>` and a `1px dotted red` border to the `<Media>`, to depict the placement of the components.

<Playground>
  <Media style={{ border: "1px dotted red" }}>
    <Media.Item align="left" style={{ border: "1px solid black" }}>
      <code>left</code>
    </Media.Item>
    <Media.Item align="content" style={{ border: "1px solid black" }}>
      <code>content</code>
    </Media.Item>
    <Media.Item align="right" style={{ border: "1px solid black" }}>
      <code>right</code>
    </Media.Item>
  </Media>
</Playground>

You can include _any_ other **rbx** component, like `<Input>`s, `<Textarea>`s, `<Icon>`s, `<Button>`s ... or even a `<Navbar>`.

<Playground>
  <Media>
    <Media.Item as="figure" align="left">
      <Image.Container as="p" size={64}>
        <Image src="https://bulma.io/images/placeholders/128x128.png" />
      </Image.Container>
    </Media.Item>
    <Media.Item align="content">
      <Field>
        <Control>
          <Textarea placeholder="Add a comment..." />
        </Control>
      </Field>
      <Level>
        <Level.Item align="left">
          <Level.Item>
            <Button color="info" type="submit">
              Submit
            </Button>
          </Level.Item>
        </Level.Item>
        <Level.Item align="right">
          <Level.Item>
            <Label>
              <Checkbox /> Press enter to submit
            </Label>
          </Level.Item>
        </Level.Item>
      </Level>
    </Media.Item>
  </Media>
</Playground>
