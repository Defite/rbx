# Dropdown

An interactive **dropdown menu** for discoverable content.

<Content>
  <p>
    The <code>{"<Dropdown>"}</code> component is a container for a dropdown
    button and a dropdown menu.
  </p>
  <ul>
    <li>
      <code>{"<Dropdown>"}</code> the <strong>main</strong> container
      <ul>
        <li>
          <code>{"<Dropdown.Trigger>"}</code> the container for a{" "}
          <code>{"<Button>"}</code>
        </li>
        <li>
          <code>{"<Dropdown.Menu>"}</code> the toggable menu,{" "}
          <strong>hidden</strong> by default
          <ul>
            <li>
              <code>{"<Dropdown.Content>"}</code> the dropdown{" "}
              <strong>box</strong>, with a white background and a shadow
              <ul>
                <li>
                  <code>{"<Dropdown.Item>"}</code> each{" "}
                  <strong>single item</strong> of the{" "}
                  <code>{"<Dropdown>"}</code>, which can either be a{" "}
                  <code>a</code> or a <code>div</code>
                </li>
                <li>
                  <code>{"<Dropdown.Divider>"}</code> a{" "}
                  <strong>horizontal line</strong> to separate{" "}
                  <code>{"<Dropdown.Item>"}</code>s
                </li>
              </ul>
            </li>
          </ul>
        </li>
      </ul>
    </li>
  </ul>
</Content>

### Content

As with all components that leverage `<Generic>` props (more below), you can use the `as` prop of `<Dropdown.Item>` to render a `div`.

<Playground>
  <div style={{ height: "250px" }}>
    <Dropdown>
      <Dropdown.Trigger>
        <Button>
          <span>Content</span>
          <Icon size="small">
            <FontAwesomeIcon icon={faAngleDown} />
          </Icon>
        </Button>
      </Dropdown.Trigger>
      <Dropdown.Menu>
        <Dropdown.Content>
          <Dropdown.Item as="div">
            <p>
              You can insert <strong>any type of content</strong> within the
              dropdown menu.
            </p>
          </Dropdown.Item>
          <Dropdown.Divider />
          <Dropdown.Item as="div">
            <p>
              You simply need to use a <code>&lt;div&gt;</code> instead.
            </p>
          </Dropdown.Item>
          <Dropdown.Divider />
          <Dropdown.Item>This is a link</Dropdown.Item>
        </Dropdown.Content>
      </Dropdown.Menu>
    </Dropdown>
  </div>
</Playground>

### Hoverable

Use the `hoverable` prop on `<Dropdown>` to trigger the menu on mouse-over.

<Playground>
  <div style={{ height: "150px" }}>
    <Dropdown hoverable>
      <Dropdown.Trigger>
        <Button>
          <span>Hover me</span>
          <Icon size="small">
            <FontAwesomeIcon icon={faAngleDown} />
          </Icon>
        </Button>
      </Dropdown.Trigger>
      <Dropdown.Menu>
        <Dropdown.Content>
          <Dropdown.Item as="div">
            <p>
              You can insert <strong>any type of content</strong> within the
              dropdown menu.
            </p>
          </Dropdown.Item>
        </Dropdown.Content>
      </Dropdown.Menu>
    </Dropdown>
  </div>
</Playground>

### Managed

Use the `managed` prop on `<Dropdown>` to fully manage the dropdown using the `active` prop of the `<Dropdown>`.

<Playground>
  <div style={{ height: "150px" }}>
    <Dropdown managed active={true}>
      <Dropdown.Trigger>
        <Button>
          <span>Always open</span>
          <Icon size="small">
            <FontAwesomeIcon icon={faAngleDown} />
          </Icon>
        </Button>
      </Dropdown.Trigger>
      <Dropdown.Menu>
        <Dropdown.Content>
          <Dropdown.Item as="div">
            <p>
              Edit this playground and change <code>active={"{true}"}</code>
              to <code>active={"{false}"}</code> to close.
            </p>
          </Dropdown.Item>
        </Dropdown.Content>
      </Dropdown.Menu>
    </Dropdown>
  </div>
</Playground>

### Alignment

Use the `align` prop of `<Dropdown>` to align the menu to the left or right.

<Playground>
  <div style={{ height: "150px" }}>
    <Level breakpoint="mobile">

        <Dropdown>
          <Dropdown.Trigger>
            <Button>
              <span>Left aligned</span>
              <Icon size="small">
                <FontAwesomeIcon icon={faAngleDown} />
              </Icon>
            </Button>
          </Dropdown.Trigger>
          <Dropdown.Menu>
            <Dropdown.Content>
              <Dropdown.Item as="div">
                <p>The dropdown is <strong>left-aligned</strong> by default.</p>
              </Dropdown.Item>
            </Dropdown.Content>
          </Dropdown.Menu>
        </Dropdown>
      </Level.Item>

      <Level.Item align="right">
        <Dropdown align="right">
          <Dropdown.Trigger>
            <Button>
              <span>Right aligned</span>
              <Icon size="small">
                <FontAwesomeIcon icon={faAngleDown} />
              </Icon>
            </Button>
          </Dropdown.Trigger>
          <Dropdown.Menu>
            <Dropdown.Content>
              <Dropdown.Item as="div">
                <p>Add the <code>align="right"</code> prop to the
                <code>{'<Divider>'}</code> for a <strong>right-aligned</strong>
                dropdown.</p>
              </Dropdown.Item>
            </Dropdown.Content>
          </Dropdown.Menu>
        </Dropdown>
      </Level.Item>
    </Level>

  </div>
</Playground>

### Dropup

Use the `up` prop on `<Dropdown>` for the menu to appear **above** the `<Dropdown.Trigger>`.

<Playground>
  <div style={{ marginTop: "150px" }}>
    <Dropdown up>
      <Dropdown.Trigger>
        <Button>
          <span>Dropup button</span>
          <Icon size="small">
            <FontAwesomeIcon icon={faAngleUp} />
          </Icon>
        </Button>
      </Dropdown.Trigger>
      <Dropdown.Menu>
        <Dropdown.Content>
          <Dropdown.Item as="div">
            <p>
              You can add the <code>up</code> prop to
              <code>{"<Dropdown>"}</code> to have a dropdown menu that appears above
              the dropdown button.
            </p>
          </Dropdown.Item>
        </Dropdown.Content>
      </Dropdown.Menu>
    </Dropdown>
  </div>
</Playground>
