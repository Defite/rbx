# Level

A multi-purpose **horizontal level**, which can contain almost any other element.

<Content>
  <p>
    The <strong>structure</strong> of a <code>{"<Level>"}</code> is the
    following:
  </p>
  <ul>
    <li>
      <code>{"<Level>"}</code>: main container
      <ul>
        <li>
          <code>{"<Level.Item>"}</code> for the left or right side
          <ul>
            <li>
              <code>{"<Level.Item>"}</code> for each individual element
            </li>
          </ul>
        </li>
      </ul>
    </li>
  </ul>
  <p>
    In a <code>{"<Level.Item>"}</code>, you can then insert almost{" "}
    <em>anything</em> you want: a <code>{"<Title>"}</code>, a{" "}
    <code>{"<Button>"}</code>, an <code>{"<Input>"}</code>, or just simple text.
    No matter what elements you put inside an <strong>rbx</strong>{" "}
    <code>{"<Level>"}</code>, they will always be{" "}
    <strong>vertically centered</strong>.
  </p>
</Content>

The example below adds a `1px solid black` border to each `<Level.Item>` and a `1px dotted red` border to the `<Level>`, to depict the placement of the components.

### Centered level

If you don't use the `align` prop of `<Level.Item>`, you can include as many `<Level.Item>`s and they'll remain centered – as long as they're the **direct** descendent of the `<Level>`.

### Breakpoint

Use the `breakpoint` prop of `<Level>` to change the responsive size at which `<Level.Item>`s are vertically placed rather than horizontal.

For example, for a responsive size greater than **mobile** (up to 768px), these `<Level.Item>`s will be stacked.

<Playground>
  <Level breakpoint="fullhd">
    <Level.Item textAlign="centered">
      <div>
        <Heading>Tweets</Heading>
        <Title as="p">3,210</Title>
      </div>
    </Level.Item>
    <Level.Item textAlign="centered">
      <div>
        <Heading>Following</Heading>
        <Title as="p">210</Title>
      </div>
    </Level.Item>
    <Level.Item textAlign="centered">
      <div>
        <Heading>Followers</Heading>

    <Level.Item textAlign="centered">

        <Heading>Likes</Heading>
        <Title as="p">321K</Title>
      </div>
    </Level.Item>
  </Level>
</Playground>
