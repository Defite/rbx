# Other Components

These are utility components for the Bulma library.
Often, these components aren't very useful on their own.

### Block

The `<Block>` component is useful for creating vertical space between unrelated content.

<Playground>
  <Block>Block 1</Block>
  <Block>Block 2</Block>
</Playground>

### Delete

A versatile **delete** cross.

Frequently used with the `<Modal>` and `<Notification>` components.

<Playground>
  <Delete />
</Playground>

Use the `size` prop of `<Delete>` to change its size.

<Playground>
  {() => {
    const sizes = DELETE_DEFAULTS.sizes.map(size => ({ name: size, size }));
    sizes.splice(1, 0, { name: "-", size: undefined });

    return sizes.map(({ size }, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Delete size={size} />
      </OptionBlock>
    ))

}}

</Playground>
