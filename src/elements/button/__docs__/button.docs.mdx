# Button

The classic button, in different colors, sizes, and states.

<Playground>
  <Button.Group>
    <Button>Button</Button>
    <Button as="a">Anchor</Button>
  </Button.Group>
</Playground>

### Colors

These are the colors you can specify using the `color` prop.

When unspecified, Bulma styles a neutral button.

<Playground>
  <Button.Group>
    {DEFAULTS.colors.map(color => (
      <Button color={color} key={color}>
        {color}
      </Button>
    ))}
  </Button.Group>
</Playground>

### Sizes

These are the sizes you can specify using the `size` prop.

When unspecified, Bulma styles a `medium` button.

<Playground>
  <Button.Group>
    {BUTTON_DEFAULTS.sizes.map(size => (
      <Button size={size} key={size}>
        {size}
      </Button>
    ))}
  </Button.Group>
</Playground>

Use the `size` prop on `<Button.Group>` to change the size of **multiple buttons** at once.

<Playground>
  <Button.Group size="medium">
    <Button>All</Button>
    <Button>Medium</Button>
    <Button>Size</Button>
  </Button.Group>
</Playground>

Use the `size` prop on `<Button>` to change the size of a **subset of buttons** in a `<Button.Group>` with a set `size` prop.

<Playground>
  <Button.Group size="small">
    <Button>small</Button>
    <Button>small</Button>
    <Button>small</Button>
    <Button size="normal">normal</Button>
    <Button>small</Button>
  </Button.Group>
</Playground>

### Groups

Group buttons together on a single line using the `<Button.Group>` component.

<Playground>
  <Button.Group>
    <Button color="link">Save Changes</Button>
    <Button>Cancel</Button>
    <Button color="danger">Delete post</Button>
  </Button.Group>
</Playground>

A long list of buttons will be wrapped while keeping the buttons evenly spaced.

<Playground>
  <Button.Group>
    {[
      "One",
      "Two",
      "Three",
      "Four",
      "Five",
      "Six",
      "Seven",
      "Eight",
      "Nine",
      "Ten",
      "Eleven",
      "Twelve",
      "Thirteen",
      "Fourteen",
      "Fifteen",
      "Sixteen",
      "Seventeen",
      "Eighteen",
      "Nineteen",
      "Twenty",
    ].map(name => (
      <Button key={name}>{name}</Button>
    ))}
  </Button.Group>
</Playground>

### Alignment

Use the `align` prop to align a button group.

<Playground>
  {() =>
    [undefined, ...BUTTON_GROUP_DEFAULTS.alignments].map((align, i) => (
      <OptionBlock props={{ align }} index={i} key={i}>
        <Button.Group hasAddons align={align}>
          <Button>Yes</Button>
          <Button>Maybe</Button>
          <Button>No</Button>
        </Button.Group>
      </OptionBlock>
    ))
  }
</Playground>

### Customization

Individual buttons can be tailored with different styles.
You should use the `selected` prop of the button to ensure it's _above_ it's siblings.

<Playground>
  <Button.Group hasAddons>
    <Button color="success" selected>
      Yes
    </Button>
    <Button>Maybe</Button>
    <Button>No</Button>
  </Button.Group>

<Button.Group hasAddons>
  <Button>Yes</Button>
  <Button color="info" selected>
    Maybe
  </Button>
  <Button>No</Button>
</Button.Group>

  <Button.Group hasAddons>
    <Button>Yes</Button>
    <Button>Maybe</Button>
    <Button color="danger">No</Button>
  </Button.Group>
</Playground>
