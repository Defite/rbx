# Select

The browser built-in **select dropdown**, styled accordingly.

The `<Select>` Component includes a wrapping container, `<Select.Container>`.

<Playground>
  <Select.Container>
    <Select>
      <Select.Option>Select dropdown</Select.Option>
      <Select.Option>With options</Select.Option>
    </Select>
  </Select.Container>
</Playground>

### Multiple select

Use the `multiple` and `size` props of `<Select>` to enable a multiple select.

<Playground>
  <Select.Container>
    <Select multiple size={4}>
      <Select.Option value="Argentina">Argentina</Select.Option>
      <Select.Option value="Bolivia">Bolivia</Select.Option>
      <Select.Option value="Brazil">Brazil</Select.Option>
      <Select.Option value="Chile">Chile</Select.Option>
      <Select.Option value="Colombia">Colombia</Select.Option>
      <Select.Option value="Ecuador">Ecuador</Select.Option>
      <Select.Option value="Guyana">Guyana</Select.Option>
      <Select.Option value="Paraguay">Paraguay</Select.Option>

</Playground>

### Colors

Use the `color` prop of `<Select.Container>` to change the component's color.

<Playground>
  {() =>
    DEFAULTS.colors.map((color, i) => (
      <OptionBlock props={{ color }} index={i} key={i}>
        <Field>
          <Control>
            <Select.Container color={color}>
              <Select>
                <Select.Option>{color} select dropdown</Select.Option>
                <Select.Option>With options</Select.Option>
              </Select>
            </Select.Container>
          </Control>
        </Field>
      </OptionBlock>
    ))
  }
</Playground>

### Sizes

Use the `size` prop of `<Select.Container>` to change the component's size.

<Playground>
  {() => {
    const sizes = SELECT_CONTAINER_DEFAULTS.sizes.map(size => ({ name: size, size}));
    sizes.splice(1, 0, { name: "normal", size: undefined });

    return sizes.map(({ name, size }, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Field>
          <Control>
            <Select.Container size={size}>
              <Select>
                <Select.Option>{name} select dropdown</Select.Option>
                <Select.Option>With options</Select.Option>
              </Select>
            </Select.Container>
          </Control>
        </Field>
      </OptionBlock>
    ))

}}

</Playground>

### States

Use the `state` prop of `<Select.Container>` to change the component's state.

<Playground>
  {() => {
    const states = [
      { name: "normal", state: undefined },
      ...SELECT_CONTAINER_DEFAULTS.states.map(state => ({ name: state, state }))
    ];

    return states.map(({name, state}, i) => (
      <OptionBlock props={{ state }} index={i} key={i}>
        <Field>
          <Control>
            <Select.Container state={state}>
              <Select>
                <Select.Option>{name} select dropdown</Select.Option>
                <Select.Option>With options</Select.Option>
              </Select>
            </Select.Container>
          </Control>
        </Field>
      </OptionBlock>
    ))

}}

</Playground>

### Icons

Use the `iconLeft` modifier on the `<Control>` holding the `<Select.Container>` to add an icon.
You also need to set the `align` property of the `<Icon>` to `left`.

The size of the `<Select>` will define the size of the `<Icon>`.

<Playground>

    <Control iconLeft>
      <Select.Container size="small">

          <Select.Option>Country</Select.Option>
          <Select.Option>Select dropdown</Select.Option>
          <Select.Option>With options</Select.Option>
        </Select>
      </Select.Container>
      <Icon size="small" align="left">
        <FontAwesomeIcon icon={faGlobe} />
