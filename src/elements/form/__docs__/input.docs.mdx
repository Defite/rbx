# Input

The text **input** and its variations.

<Playground>
  <Input type="text" placeholder="Text input" />
</Playground>

### Colors

Use the `color` prop to change the `<Input>`'s color.

<Playground>
  {() =>
    DEFAULTS.colors.map((color, i) => (
      <OptionBlock props={{ color }} index={i} key={i}>
        <Field>
          <Control>
            <Input type="text" color={color} placeholder={`${color} input`} />
          </Control>
        </Field>
      </OptionBlock>
    ))
  }
</Playground>

### Sizes

Use the `size` prop to change the `<Input>`'s size.

<Playground>
  {() => {
    const sizes = INPUT_DEFAULTS.sizes.map(size => ({ name: size, size }));
    sizes.splice(1, 0, { name: "normal", size: undefined });

    return sizes.map(({ name, size}, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Field>
          <Control>
            <Input type="text" size={size} placeholder={`${name} input`} />
          </Control>
        </Field>
      </OptionBlock>
    ))

}}

</Playground>

### Styles

Use the `rounded` prop to round the edges of the `<Input>`.

<Playground>
  <Input type="text" rounded placeholder="rounded input" />
</Playground>

### States

Use the `state` prop to change the `<Input>`'s state.

<Playground>
  {() => {
    const states = INPUT_DEFAULTS.states.map(state => ({ name: state, state }));
    states.splice(0, 0, { name: "normal", size: undefined });

    return states.map(({ name, state}, i) => (
      <OptionBlock props={{ state }} index={i} key={i}>
        <Field>
          <Control>
            <Input type="text" state={state} placeholder={`${name} input`} />
          </Control>
        </Field>
      </OptionBlock>
    ))

}}

</Playground>

Use the `loading` prop on the `<Control>` that contains the `<Input>` to get a loading spinner.
It's size can be adjusted by passing an appropriate `size` prop to the `<Control>` and `<Input>`.

<Playground>
  <Field>
    <Control size="small" loading>
      <Input type="text" size="small" placeholder="small loading input" />
    </Control>
  </Field>

<Field>

    <Input type="text" placeholder="normal loading input" />
  </Control>
</Field>

<Field>
  <Control size="medium" loading>

  </Control>
</Field>

  <Field>
    <Control size="large" loading>
      <Input type="text" size="large" placeholder="large loading input" />

  </Field>
</Playground>

### Disabled

Use the standard `disabled` prop on the `<Input>` to disable it.

<Playground>
  <Control>
    <Input type="text" placeholder="Disabled" disabled />
  </Control>
</Playground>

### Read-only

Use the `readOnly` prop to mark an `<Input>` as read-only.
It's not editable, and has no shadow.

<Playground>
  <Control>
    <Input type="text" placeholder="This text is read-only" readOnly />
  </Control>
</Playground>

### Static

Use the `static` prop along with `readOnly` to mark an `<Input>` to remove the background, border, shadow and horizontal padding.
The vertical spacing will be maintained, so you can align the input in any context – like a horizontal form.

<Playground>
  <Field horizontal>
    <Field.Label size="normal">
      <Label>From</Label>
    </Field.Label>
    <Field.Body>
      <Control>
        <Input type="email" value="me@example.com" readOnly static />
      </Control>
    </Field.Body>
  </Field>

  <Field horizontal>
    <Field.Label>
      <Label>To</Label>
    </Field.Label>
    <Field.Body>
      <Control>
        <Input type="email" placeholder="Recipient email" />
      </Control>
    </Field.Body>
  </Field>
</Playground>

### With icons

Use the `iconLeft` or `iconRight` prop on the `<Control>` that contains an `<Input>` to add icons.
You'll also need to use the `align` property on the `<Icon>`.

<Playground>
  <Field>
    <Control iconLeft iconRight>
      <Input type="email" placeholder="Email" />
      <Icon align="left">
        <FontAwesomeIcon icon={faEnvelope} />
      </Icon>
      <Icon align="right">
        <FontAwesomeIcon icon={faCheck} />
      </Icon>
    </Control>
  </Field>

  <Field>
    <Control iconLeft>
      <Input type="password" placeholder="Password" />
      <Icon align="left">
        <FontAwesomeIcon icon={faLock} />
      </Icon>
    </Control>
  </Field>
</Playground>

If the control contains an icon, it will remain **centered** regardless of the size foo the `<Input>` or `<Icon>`.
