# Icon

**rbx** is compatible with all icon font libraries:
[Font Awesome 5](https://fontawesome.com),
[Font Awesome 4](http://fontawesome.io/),
[Material Design Icons](https://materialdesignicons.com/),
[Open Iconic](https://useiconic.com/open),
[Ionicons](http://ionicons.com/),
etc.

The `<Icon>` component is a container for any type of **icon font**.
If the icon takes a few seconds to load, the `<Icon>` container provides a reliable square that will prevent page jumps as loading occurs.

The examples here use **Font Awesome** icons.

Here is basic usage, with the `<Icon>`'s `backgroundColor` prop set to highlight the icon.

The `backgroundColor` prop is a `<Generic>` prop that the `<Icon>` component leverages.
Read more about this in the props section below.

<Playground>
  <Icon backgroundColor="warning">
    <FontAwesomeIcon icon={faHome} />
  </Icon>
</Playground>

### Colors

Use the `color` prop to add color to an icon.

<Playground>
  {() => {
    const permutations = [
      { color: "info", icon: faInfoCircle },
      { color: "success", icon: faCheckSquare },
      { color: "warning", icon: faExclamationTriangle },
      { color: "danger", icon: faBan },
    ];
    return permutations.map(({ color, icon }) => (
      <Icon color={color} key={color}>
        <FontAwesomeIcon icon={icon} />
      </Icon>
    ));
  }}
</Playground>
