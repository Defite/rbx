# Tag

Small **tag labels** to insert anywhere.

<Playground>
  <Tag>Tag label</Tag>
</Playground>

### Colors

Use the `color` prop of a `<Tag>` for variety.

<Playground>
  <Tag.Group>
    {DEFAULTS.colors.map(color => (
      <Tag key={color} color={color}>
        {color}
      </Tag>
    ))}
  </Tag.Group>
</Playground>

### Sizes

Use the `size` prop on `<Tag>` to change its size.

<Playground>
  {() =>
    TAG_DEFAULTS.sizes.map((size, i) => (
      <OptionBlock props={{ size }} index={i} key={i}>
        <Tag size={size}>{size}</Tag>
      </OptionBlock>
    ))
  }
</Playground>

Use the `size` prop on `<Tag.Group>` to change the size of all `<Tag>`s in that group.

<Playground>
  {() => {
    const sizes = [
      { name: 'default', size: undefined },
      ...TAG_GROUP_DEFAULTS.sizes.map(size => ({ name: size, size }))
    ]

    return sizes.map(({name, size}, i) => (
      <OptionBlock key={i} index={i} props={{ size }}>
        <Tag.Group size={size}>
          <Tag>All</Tag>
          <Tag>{name}</Tag>
          <Tag>Size</Tag>
        </Tag.Group>
      </OptionBlock>
    ))

}}

</Playground>

Use the `size` prop on individual `<Tag>` elements to change their size within a `<Tag.Group>` that has a set `size` prop.

<Playground>
  <Tag.Group size="medium">
    <Tag>medium</Tag>
    <Tag size="normal">normal</Tag>
    <Tag>medium</Tag>
    <Tag size="large">large</Tag>
    <Tag>medium</Tag>
  </Tag.Group>
</Playground>

### Rounding

Use the `rounded` prop on `<Tag>` to round the edges.

<Playground>
  <Tag rounded>Rounded</Tag>
</Playground>

### Delete

Use the `delete` prop on `<Tag>` to make it a delete button.

<Playground>
  <Tag delete />
</Playground>

Or use a `<Delete>` component to add a delete button in the tag.

<Playground>
  <Block>
    <Tag color="success">
      Bar
      <Delete size="small" />
    </Tag>
  </Block>

<Block>
  <Tag color="warning" size="medium">
    Hello
    <Delete size="small" />
  </Tag>
</Block>

  <Block>
    <Tag color="danger" size="large">
      World
      <Delete />
    </Tag>
  </Block>
</Playground>

### List of tags

Use the `<Tag.Group>` component to group tags together in a list.

<Playground>
  <Tag.Group>
    <Tag>One</Tag>
    <Tag>Two</Tag>
    <Tag>Three</Tag>
  </Tag.Group>
</Playground>

If the list is long, it will automatically wrap on multiple lines, while keeping the tags evenly spaced.

<Playground>
  <Tag.Group>
    {[
      "One",
      "Two",
      "Three",
      "Four",
      "Five",
      "Six",
      "Seven",
      "Eight",
      "Nine",
      "Ten",
      "Eleven",
      "Twelve",
      "Thirteen",
      "Fourteen",
      "Fifteen",
      "Sixteen",
      "Seventeen",
      "Eighteen",
      "Nineteen",
      "Twenty",
    ].map(name => (
      <Tag key={name}>{name}</Tag>
    ))}
  </Tag.Group>
</Playground>

### Attach tags

Use the `gapless` prop on `<Tag.Group>` to attach `<Tag>`s togther.

<Playground>
  <Tag.Group gapless>
    <Tag>Package</Tag>
    <Tag color="info">rbx</Tag>
    <Tag color="primary">Bulma</Tag>
  </Tag.Group>
</Playground>

You can attach a **text** and a **delete** tag together.

<Playground>
  <Tag.Group gapless>
    <Tag color="danger">Alex Smith</Tag>
    <Tag delete />
  </Tag.Group>
</Playground>

To attach `gapless` `<Tag.Group>`s together, use the `<Field>` component to wrap them and provide the `<Field>` component the `kind="group"` and `multiline` props.

<Playground>
  <Field kind="group" multiline>
    <Control>
      <Tag.Group gapless>
        <Tag color="dark">npm</Tag>
        <Tag color="info">2.0.0</Tag>
      </Tag.Group>
    </Control>
    <Control>
      <Tag.Group gapless>
        <Tag color="dark">build</Tag>
        <Tag color="success">passing</Tag>
      </Tag.Group>
    </Control>
    <Control>
      <Tag.Group gapless>
        <Tag color="dark">chat</Tag>
        <Tag color="primary">#reactiflux</Tag>
      </Tag.Group>
    </Control>
  </Field>
</Playground>

This can be useful for a long list of **blog tags**.

<Playground>
  <Field multiline kind="group">
    {[
      "Technology",
      "CSS",
      "Flexbox",
      "Web Design",
      "Open Source",
      "Community",
      "Documentation",
    ].map(name => (
      <Control key={name}>
        <Tag.Group gapless>
          <Tag color="link">{name}</Tag>
          <Tag delete />
        </Tag.Group>
      </Control>
    ))}
  </Field>
</Playground>
