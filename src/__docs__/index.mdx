### The Comprehensive _Bulma UI_ Framework for _React_

[![Build Status](https://travis-ci.org/dfee/rbx.svg?branch=master)](https://travis-ci.org/dfee/rbx)
[![Coverage Status](https://coveralls.io/repos/github/dfee/rbx/badge.svg?branch=master)](https://coveralls.io/github/dfee/rbx?branch=master)
[![Release Version](https://img.shields.io/github/release/dfee/rbx.svg)](https://github.com/dfee/rbx)
[![Npm Downloads](https://img.shields.io/npm/dm/rbx.svg)](https://www.npmjs.com/package/rbx)

**rbx** is a complete implementation of the **latest** [Bulma UI Framework](http://bulma.io/) for [React](http://reactjs.org/).
It's designed from the ground up to be robust _and_ easy to use.

<Block>
  <Title as="h3" size={3}>
    Getting started
  </Title>
  <p>
    After you <code>npm install rbx</code>, you can simply import the CSS styles
    and components from <strong>rbx</strong>:
  </p>
</Block>

```
import React from "react";
import ReactDOM from "react-dom";

import "rbx/index.css";
import { Content } from "rbx";

function App() {
  return (
    <Content>
      Hello world from <a href="https://github.com/dfee/rbx">rbx</a>!
    </Content>
  );
}

const rootElement = document.getElementById("root");
ReactDOM.render(<App />, rootElement);
```
